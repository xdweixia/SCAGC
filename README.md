# SCAGC
The Pytorch implementation demo for our TMM article: Self-consistent Contrastive Attributed Graph Clustering with Pseudo-label Prompt.

## Requirements
- Python 3.6
- PyTorch 1.6.0
- PyTorch Geometric 1.6.1

## Run
```
python run_cluster.py
```
